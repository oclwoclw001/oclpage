# Neon Stack

一个纯静态的俄罗斯方块小游戏，适合直接部署到 GitHub Pages。

## 功能

- 10x20 标准棋盘
- 7-bag 随机
- Hold / Next Queue
- Ghost Piece
- 软降 / 硬降
- 基础墙踢旋转
- 分数、等级、消行统计
- 暂停、重开、游戏结束覆盖层
- 键盘和手机触屏按钮

## 本地打开

直接双击 `index.html` 即可。

## 发布到 GitHub Pages

把这些文件推到仓库默认分支，然后在 GitHub 仓库设置里启用 Pages：

- `Settings`
- `Pages`
- `Build and deployment`
- Source 选 `Deploy from a branch`
- Branch 选默认分支 `/ root`

如果仓库已经是 Pages 仓库，推送后通常会自动更新。
